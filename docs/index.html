Freon Language Workbench (Work in Progress)

This is the future homepage of Freon, a workbench for developing Domain Specific Languages (DSL) that work natively on the web.

This project is a direct descendent of ProjectIt, a projectional editor for the web.

Because the nature of ProjeectIt has changed from a tool to create a projectional editor to a tool to create a full DSL, it was decided to change the name of the project.
